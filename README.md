# MultiInterface Sender

MultiInterface Sender is an app realized to allow people sending each other big files through multiple interfaces at the same time.
Mobile data, wireless and bluetooth can be used togeather to speed up sending files between pc and mobile, besides mobile and mobile.

This app is under developing, along with a client desktop, and will be officially launched in few weeks.
